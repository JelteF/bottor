bottor
======

A friendly python botnet

(moahaha)
## Requirements
* Python 3
* Virtualenv

## Tracker Setup
* `cd tracker`
* `virtualenv -p python3 venv`
* `. venv/bin/activate`
* `pip install -r requirements.txt`


## Client Setup
* `cd client`
* `virtualenv -p python3 venv`
* `. venv/bin/activate`
* `pip install -r requirements.txt`
